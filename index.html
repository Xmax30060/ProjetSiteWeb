Définitions :

Le langage HTML (HyperText Markup Language) permet de structurer et de mettre en forme les éléments d’une page Web. Il s’écrit avec des balises qui fonctionnent toujours par deux : la balise d’ouverture et la balise de fermeture.

CSS est l'acronyme de « Cascading Style Sheets » ce qui signifie « feuille de style en cascade ». Le CSS correspond à un langage informatique permettant de mettre en forme des pages web (HTML). Ce langage est donc composé des fameuses « feuilles de style en cascade » également appelées fichiers CSS.

Biographie de son inventeur

Tim Berners-Lee, né le 8 juin 1955 à Londres, est un informaticien britannique, inventeur du World Wide Web (WWW). Après des études en physique à l'Université d'Oxford, il travaille au CERN dans les années 1990, où il développe les technologies fondamentales du Web : les URL, le protocole HTTP et le langage HTML. Il fonde le World Wide Web Consortium (W3C) en 1994 pour superviser le développement du Web. En 2004, il est fait chevalier par la reine Elizabeth II pour ses contributions à l'informatique.

Index.html: Tuto Vidéo

                
                
    <!DOCTYPE html>
    <html lang="fr">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Titre de la page</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>Ceci est un exemple de code affiché en HTML.</p>
    </body>
    </html>

Style.css: Références des règles

                
                
    /* Styles de base */
    body {
        font-family: Arial, sans-serif;
        background-color: #f4f4f4;
        color: #333;
    }

    /* Style pour les titres */
    h1 {
        color: #007bff;
        text-align: center;
    }

    /* Style pour une boîte */
    .box {
        width: 300px;
        padding: 20px;
        background-color: #fff;
        border-radius: 10px;
        box-shadow: 2px 2px 10px rgba(0, 0, 0, 0.1);
    }
